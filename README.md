# API Login Python

Sistema de login desenvolvido com Python para estudo de desenvolvimento Backend.

## Tecnologias utilizadas

- Python
- FastAPI
- SQLite
- HTML
- CSS
- JavaScript

## Funcionalidades

- Login de usuários
- Cadastro de usuários
- Página de produtos
- Banco de dados SQLite

## Objetivo

Este projeto foi desenvolvido para praticar desenvolvimento Backend e compor meu portfólio como desenvolvedora.
